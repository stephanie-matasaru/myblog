# Personal Blog Web Application

A responsive, full-stack personal blog web application built with Python and Django. This project features dynamic content rendering, database integration, and a custom frontend interface.

## Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite (Default Django DB)

## Main Features
- Dynamic routing and page rendering using Django templates.
- Interactive frontend elements built with JavaScript.
- Custom styling and responsive layout using raw CSS and HTML.

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone [https://github.com/stephanie-matasaru/myblog.git](https://github.com/stephanie-matasaru/myblog.git)
   cd myblog
Install dependencies:

Bash
pip install django
# Or run: pip install -r requirements.txt (if available)
Apply database migrations:

Bash
python manage.py migrate
Start the local development server:

Bash
python manage.py runserver
Open your browser and navigate to http://127.0.0.1:8000/
